# WeClapp Manager

Web-App zur Ergänzung des WeClapp ERP/CRM Systems mit Microsoft AD Authentication und ClockIn Integration.

## Features
- Microsoft Azure AD Authentication
- WeClapp OpenAPI Integration
- Aufgabenmanagement für Mitarbeiter
- Projekt-Synchronisation aus WeClapp
- Aufgabenvorlagen-Verwaltung
- Zeiterfassung mit ClockIn Integration

## Installation
```bash
npm install
```

## Entwicklung
```bash
npm run dev
```

## Build
```bash
npm run build
```
